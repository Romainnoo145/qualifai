Loading states.

Drie waves voor generic page-load (sub-1s). Alle drie hergebruiken Klarifai brand-tokens (Sora, navy/gold, geen invented accenten). Kies de one die het beste schaalt over de hele admin.

[ 01 ] Skeleton
Placeholder shapes pulsen met shimmer. Layout staat al, content vult. Modern, low-cog. Werkt vooral als je consistent skeleton-shapes maakt per surface.
[ 02 ] Branded mark
Klarifai wordmark gecentreerd, gouden punt pulseert. Zelfde brand-cue als alle headings. Calm, premium, content fade-in op resolve. Goed voor full-page navigatie maar kan zwaar voelen voor in-page swaps.

[ 03 ] Top bar
Smalle gouden lijn bovenaan, indeterminate slide. Last seen state blijft staan (vervaagd, niet weg) zodat user context niet verliest. Subtiel, snel, dominant niet. Beste fit voor sub-1s loads.

Kies criterium
01 Skeleton — beste UX-wise als de loading-content vooraf duidelijk is (lijsten, dashboards). Vereist per-surface skeleton-shapes. Meeste werk maar meest "professioneel" gevoel.
02 Branded mark — sterkste brand-signaal. Werkt als alle laden > 200ms is. Bij sub-100ms voelt 'm onnodig (flash). Best als first-paint loader.
03 Top bar — minimaal, geen layout-shift, user verliest geen context. Werkt vrijwel altijd, ook bij hele snelle loads (geen flash). Kleinste implementatie. Mogelijke nadeel: minder "rich" gevoel.
