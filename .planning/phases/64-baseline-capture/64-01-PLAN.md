Create `scripts/capture-baseline.ts` following the established dotenv + PrismaPg pattern from `scripts/tmp-run-analysis-nedri.ts`.

The script must:

1. **Query all ProspectAnalysis records** with their associated Prospect (slug, readableSlug, companyName, domain) and ResearchRun (id, createdAt, status) included. Use `prisma.prospectAnalysis.findMany({ include: { prospect: { select: { slug: true, readableSlug: true, companyName: true, domain: true } }, researchRun: { select: { id: true, createdAt: true, status: true } } } })`.

2. **Generate output directory** at `.planning/baselines/analysis/`. Use `fs.mkdirSync(outDir, { recursive: true })`.

3. **Generate ISO timestamp** once at script start: `new Date().toISOString().replace(/[:.]/g, '-').slice(0, 19)` (e.g., `2026-04-21T14-30-00`). This timestamp is shared across all files in a single capture run.

4. **Write one JSON file per prospect** with filename pattern: `{readableSlug ?? slug}_{timestamp}.json` (e.g., `nedri-spanstaal_2026-04-21T14-30-00.json`). Use `readableSlug` when available (human-readable), fall back to `slug`.

5. **JSON file structure** (pretty-printed with 2-space indent):

```json
{
  "capturedAt": "2026-04-21T14:30:00.000Z",
  "prospect": {
    "slug": "xxx",
    "readableSlug": "nedri-spanstaal",
    "companyName": "Nedri Spanstaal BV",
    "domain": "nedri.nl"
  },
  "analysis": {
    "id": "cuid...",
    "version": "analysis-v2",
    "createdAt": "2026-03-15T...",
    "modelUsed": "gemini-2.5-pro",
    "content": { ... full ProspectAnalysis.content JSON ... },
    "inputSnapshot": { ... }
  },
  "researchRun": {
    "id": "cuid...",
    "createdAt": "2026-03-15T...",
    "status": "COMPLETED"
  }
}
```

6. **Console output**: Print each prospect slug + filename as it writes. Print total count at end. Print "No ProspectAnalysis records found" and exit 0 if none exist.

7. **Idempotency**: The timestamp in the filename ensures reruns create new files, never overwrite existing ones.

8. **No external dependencies** beyond what's already in package.json (dotenv, @prisma/adapter-pg, pg, @prisma/client, node:fs, node:path).
