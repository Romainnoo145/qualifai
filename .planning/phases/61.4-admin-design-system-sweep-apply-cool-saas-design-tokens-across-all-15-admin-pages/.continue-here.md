## Design Patterns Established

- Page title: 48px Sora 700 with gold period, max-width 1400px centered
- Section headers: label text + extending 1px line
- Toggle pills: navy active, ghost inactive (border + transparent)
- Primary CTA: gold gradient pill (rounded-full)
- Secondary: navy solid rounded-md
- Ghost: transparent + border, hover darkens border to ink
- Rows: borderless, subtle dividers, hover shifts left 2-6px
- Circular arrow: 32-36px, border, hover → navy bg + gold arrow
- Empty states: centered icon + title + description, no cards
- Choice cards: navy bg + gold title active, ghost inactive
- Status: text (not colored badges) — "Solid" navy, "Limited" rust, "Thin" muted italic
