## Decisions Made

- `admin-btn-primary` navy solid is definitive — gold gradient exists only in `btn-pill-*` classes (brochure surface). This was a pre-locked design decision already documented; the globals.css was simply out of sync.
- `StatusBadge` interface uses `status: string` (not a union type) to avoid tight coupling with Prisma enums and to remain flexible for future status additions without touching this component.
- `QuoteStatusBadge` callers all pass `status={...}` as a string prop — type widens cleanly from `QuoteStatus` to `string`, zero breaking changes.
