<objective>
Make the master analyzer survive transient Gemini API failures that caused the Marfa pipeline to silently drop its narrative analysis during Phase 61 smoke testing. Add a retry layer with exponential backoff, a fallback from `gemini-2.5-pro` to `gemini-2.5-flash` on exhaustion, and persist failure state AND model-used state on the `Prospect` row so the UI (Plan 04) can render a friendly status — including a "fallback used" warning when the pipeline silently degraded to flash.

Purpose: Resilience — AI pipelines currently fail loudly with stack traces on HTTP 503, which is indistinguishable from a real bug. A transient spike must not lose the analysis. AND: when fallback kicks in, the UI must surface that signal (ROADMAP SC #2) instead of showing a plain green checkmark.

Output:

- `callGeminiWithRetry` helper wrapping both `generateContent` callsites in `master-analyzer.ts` — returns `GeminiCallResult = {result, fallbackUsed, modelUsed, attempts}`, NOT a plain `GenerateContentResult`
- New `lastAnalysisError` + `lastAnalysisAttemptedAt` + `lastAnalysisModelUsed` columns on `Prospect`, added via an additive Prisma migration
- `recordAnalysisSuccess(db, prospectId, modelUsed)` helper (replaces `clearAnalysisFailure`): writes `lastAnalysisModelUsed` with the actual model that succeeded, clears `lastAnalysisError`, stamps `lastAnalysisAttemptedAt`
- `recordAnalysisFailure(db, prospectId, errorMessage)` helper for the hard-failure path
- Top-level `generateNarrativeAnalysis` / `generateKlarifaiNarrativeAnalysis` propagate `fallbackUsed` + `modelUsed` on their return values so the tRPC mutation (Plan 03) can forward them to the UI
- Unit tests that mock the Gemini SDK and assert: (a) 3 retries on 503, (b) fallback to flash with `fallbackUsed=true`, (c) immediate bubble on non-retryable errors, (d) success returns `fallbackUsed=false` on first-try success

       ```typescript
       // ---------------------------------------------------------------------------
       // Retry + fallback layer (Phase 61.1 POLISH-01/02 + SC #2)
       // PHASE 61.1 POLISH: retry layer inlined — candidate for extraction to lib/analysis/retry.ts
       // ---------------------------------------------------------------------------

       /**
        * Return envelope for callGeminiWithRetry. Callers MUST destructure this —
        * the `fallbackUsed` flag is the fidelity signal that drives the Plan 04 UI
        * fallback-used warning (ROADMAP SC #2).
        */
       export type GeminiCallResult = {
         result: GenerateContentResult;
         fallbackUsed: boolean;
         modelUsed: 'gemini-2.5-pro' | 'gemini-2.5-flash';
         attempts: number;
       };

       const RETRYABLE_PATTERNS = [
         /5\d\d/,                    // any HTTP 5xx in the message
         /Service Unavailable/i,
         /quota/i,
         /429/,
         /rate.?limit/i,
       ];

       function isRetryableGeminiError(error: unknown): boolean {
         const message =
           error instanceof Error ? error.message : String(error);
         return RETRYABLE_PATTERNS.some((rx) => rx.test(message));
       }

       /**
        * Call Gemini with exponential backoff + optional fallback model.
        *
        * Retry matrix:
        *   - 3 primary attempts, backoff 1s -> 4s -> 16s (Math.pow(4, attempt) * 1000)
        *   - 1 fallback attempt if opts.fallbackModel provided
        *   - Non-retryable errors (4xx except 429) bubble immediately
        *
        * Returns a GeminiCallResult envelope so callers can propagate the
        * `fallbackUsed` + `modelUsed` signals to the UI layer (Plan 04).
        *
        * Exported for unit testing.
        */
       export async function callGeminiWithRetry(
         primaryModel: GenerativeModel,
         prompt: string,
         opts: {
           modelName: 'gemini-2.5-pro' | 'gemini-2.5-flash';
           fallbackModel?: GenerativeModel;
           fallbackModelName?: 'gemini-2.5-pro' | 'gemini-2.5-flash';
         },
       ): Promise<GeminiCallResult> {
         const maxAttempts = 3;
         let lastError: unknown = null;
         let attemptsConsumed = 0;

         for (let attempt = 0; attempt < maxAttempts; attempt++) {
           attemptsConsumed = attempt + 1;
           try {
             const result = await primaryModel.generateContent(prompt);
             return {
               result,
               fallbackUsed: false,
               modelUsed: opts.modelName,
               attempts: attemptsConsumed,
             };
           } catch (error) {
             lastError = error;
             if (!isRetryableGeminiError(error)) {
               throw error;
             }
             if (attempt < maxAttempts - 1) {
               const delay = Math.pow(4, attempt) * 1000; // 1s, 4s, 16s
               console.warn(
                 `[master-analyzer] ${opts.modelName} attempt ${attempt + 1}/${maxAttempts} failed (retryable), waiting ${delay}ms: ${error instanceof Error ? error.message.slice(0, 200) : String(error)}`,
               );
               await new Promise((resolve) => setTimeout(resolve, delay));
             }
           }
         }

         if (opts.fallbackModel && opts.fallbackModelName) {
           console.warn(
             `[master-analyzer] Falling back to ${opts.fallbackModelName} after ${maxAttempts} retries on ${opts.modelName}`,
           );
           try {
             const result = await opts.fallbackModel.generateContent(prompt);
             return {
               result,
               fallbackUsed: true,
               modelUsed: opts.fallbackModelName,
               attempts: attemptsConsumed,
             };
           } catch (fallbackError) {
             throw new Error(
               `[master-analyzer] Failed after ${maxAttempts} primary + 1 fallback attempt. Primary=${opts.modelName} Fallback=${opts.fallbackModelName}. Last primary error: ${lastError instanceof Error ? lastError.message.slice(0, 200) : String(lastError)}. Fallback error: ${fallbackError instanceof Error ? fallbackError.message.slice(0, 200) : String(fallbackError)}`,
             );
           }
         }

         throw new Error(
           `[master-analyzer] Failed after ${maxAttempts} attempts on ${opts.modelName} (no fallback configured). Last error: ${lastError instanceof Error ? lastError.message.slice(0, 200) : String(lastError)}`,
         );
       }
       ```
