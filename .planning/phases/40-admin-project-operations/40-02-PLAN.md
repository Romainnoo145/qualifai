# Phase 40-02 Plan — Partnership Conversion Instrumentation

Expose conversion visibility from Atlantis discover readiness signals to campaign actions, grouped by SPV.

## Scope

1. CTA instrumentation

- Track Atlantis-specific CTA interactions (meeting/context-pack/validation-call)
- Persist readiness tier + trigger metadata on event payloads
- Keep existing wizard tracking behavior unchanged

2. Admin conversion view

- Add SPV-level funnel block for Atlantis:

  - CTA clicked
  - call booked
  - campaign/prospect engaged
- Add readiness tier breakdown per SPV

3. Campaign handoff hooks

- Surface quick actions from high-readiness Atlantis prospects into campaign workflow
- Preserve existing campaign pipeline for Klarifai

2. Validate SPV funnel counters on admin dashboard
3. Confirm no cross-project leakage in metrics queries
4. Smoke test manual campaign handoff from Atlantis prospect
