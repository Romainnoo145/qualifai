The admin surface must feel like a sales automation SaaS (Linear, Attio) — NOT an editorial magazine or interior design app.

**Why:** The "Editorial" paper/cream aesthetic (#f5f2ea) with IBM Plex Mono felt like a design magazine. User explicitly rejected it mid-session on 2026-04-17.

**How to apply:**

- Canvas: cool near-white (#fafafa), zinc-based neutrals for borders/surfaces
- Font: Sora only — no IBM Plex Mono anywhere in admin. Use tabular-nums for number alignment.
- Corners: soft rounded (8px buttons, 12-20px cards), not hard 4px rectangles
- Borders: minimal — prefer subtle divider lines or bg fills over border boxes
- Gold: bright yellow family (#ebcb4b), never muddy dark gold
- No glass-card visual weight — clean, flat, spacious
