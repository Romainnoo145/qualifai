---
name: No brown/dark gold in admin UI
description: User strongly dislikes dark/muddy gold tones (#c79a1f) — admin must use bright yellow gold (#ebcb4b)
type: feedback
---

Never use dark/brown gold tones like #c79a1f or #a87f15 in the admin surface. The user finds these muddy and off-brand against the cool white canvas.

**Why:** The admin is a sales automation tool, not an interior design magazine. Dark gold reads as "luxury lifestyle" which is wrong for the product.

**How to apply:** Always use the bright gold family: --color-gold (#ebcb4b), --color-gold-hi (#f4d95a), hover (#d4b43b). Check globals.css for current values before introducing any gold/yellow.
